    <h2>各項製作心得</h2>
    <h3>A 巧克力戚風蛋糕捲</h3>
    <p>為避免表皮破裂影響外觀分數</p>
    <ol>
      <li>注意捲起時動作</li>
      <li>內部塗抹需均勻，但在中間及兩側可多塗一些</li>
      <li>表皮可用190度多烘烤10分鐘使表面不易黏在紙上</li>
    </ol>
    <h3>B 大理石蛋糕</h3>
    <ul>
      <li>黑白麵糊不可過度拌勻</li>
      <li>約考15分鐘後可割線</li>
    </ul>
    <p>大理石蛋糕是天使題目，遇到要好好把握</p>
